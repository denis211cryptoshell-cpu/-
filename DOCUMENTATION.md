## ⚙️ Система Кэширования

### Двухуровневое Кэширование

**L1 — Local (In-Memory)**
- Скорость: ~0.1ms
- Хранение: оперативная память процесса
- Для: часто запрашиваемые данные

**L2 — Redis**
- Скорость: ~1-5ms
- Хранение: внешний Redis сервер
- Для: персистентность и масштабирование

### Что Кэшируется

| Данные | TTL | Уровень |
|--------|-----|---------|
| Текст разделов | 5 мин | L1 + L2 |
| Фото (file_id) | 5 мин | L1 + L2 |
| Проверка подписки | 1 мин | L1 + L2 |
| Пригласительные ссылки | 2 часа | L1 |

### Преимущества

- ⚡ **Скорость** — мгновенный отклик
- 💾 **Персистентность** — кэш не теряется при рестарте
- 🔄 **Масштабирование** — работа с несколькими инстансами
- 🎯 **Гибкость** — переключение между L1/L2 через .env
